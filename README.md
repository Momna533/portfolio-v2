🌐 Portfolio Website
📖 Overview
This is my personal portfolio website showcasing my work as a Frontend Web Developer. It includes recruiter-ready projects, landing pages, and business websites designed for small businesses and niche clients. The portfolio emphasizes speed, mobile-friendliness, and credibility through proof-driven design.

🎯 Features
Responsive design (optimized for mobile, tablet, and desktop)

Modern UI with clean hero and about sections

Project showcase with live demos and source code links

Contact form with validation and error handling

Optimized for performance and accessibility

🛠️ Tech Stack
HTML5

CSS3 (Flexbox, Grid, Animations)

JavaScript (Validation, Error Handling)

Optional: Frameworks (React, Vue, or plain JS depending on your build)
